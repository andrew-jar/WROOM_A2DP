WROOM_A2DP - flashing instructions

Author: A. Jaroszuk

Poprawki: (28.06.2026)
  -  BEZPIECZNIK STARTOWY (Dla synchronizacji z S3) 10s.
  -  Bezpiecznik RF -czas na wygaszenie rejestrów radiowych i wyczyszczenie wewnętrznego bufora RF przed próbą zestawienia sesji A2DP TX.


Folder contents:
- bootloader.bin
- partitions.bin
- app.bin

Flash addresses (ESP32-WROOM):
- bootloader.bin  -> 0x1000
- partitions.bin  -> 0x8000
- app.bin         -> 0x10000

Requirements:
1) Python + esptool installed
2) USB-UART driver for the board
3) Knowledge of the COM port number (for example COM5)

Install esptool:
python -m pip install esptool

Full flash (recommended):
python -m esptool --chip esp32 --port COM5 --baud 921600 write_flash -z --flash_mode dio --flash_freq 40m --flash_size detect 0x1000 bootloader.bin 0x8000 partitions.bin 0x10000 app.bin

Quick update of the application only (when bootloader/partitions already match):
python -m esptool --chip esp32 --port COM5 --baud 921600 write_flash -z 0x10000 app.bin

Optional erase before full flash:
python -m esptool --chip esp32 --port COM5 erase_flash

Practical steps:
1) Open a terminal in the release folder.
2) Run the full flash command (replace COM5 with your port).
3) If flashing does not start, put the board into boot mode (BOOT while resetting).
4) After flashing, open the serial monitor at 115200 baud and check for: READY WROOM-BT-TX v1.6.1 (CB-reset enabled)

Notes:
- If you use a different ESP32 module, the safest option is still a full flash with all three files.
- These files come from one consistent build (same compile time).
