WROOM_A2DP - instrukcja flashowania

Autor: A. Jaroszuk

Poprawki: (28.06.2026)
  -  BEZPIECZNIK STARTOWY (Dla synchronizacji z S3) 10s.
  -  Bezpiecznik RF -czas na wygaszenie rejestrów radiowych i wyczyszczenie wewnętrznego bufora RF przed próbą zestawienia sesji A2DP TX.


Zawartosc folderu:
- bootloader.bin
- partitions.bin
- app.bin

Adresy flash (ESP32-WROOM):
- bootloader.bin  -> 0x1000
- partitions.bin  -> 0x8000
- app.bin         -> 0x10000

Wymagania:
1) Zainstalowany Python + esptool
2) Sterownik USB-UART do plytki
3) Znajomosc portu COM (np. COM5)

Instalacja esptool:
python -m pip install esptool

Pelne wgranie (zalecane):
python -m esptool --chip esp32 --port COM5 --baud 921600 write_flash -z --flash_mode dio --flash_freq 40m --flash_size detect 0x1000 bootloader.bin 0x8000 partitions.bin 0x10000 app.bin

Szybka aktualizacja samej aplikacji (gdy bootloader/partitions juz pasuja):
python -m esptool --chip esp32 --port COM5 --baud 921600 write_flash -z 0x10000 app.bin

Opcjonalnie czyszczenie flash przed pelnym wgraniem:
python -m esptool --chip esp32 --port COM5 erase_flash

Kroki praktyczne:
1) Otworz terminal w folderze release.
2) Uruchom komende pelnego wgrania (podmien COM5 na swoj port).
3) Jesli flash nie startuje, wprowadz plytke w tryb boot (BOOT przy reset).
4) Po flashu otworz monitor portu 115200 i sprawdz napis READY WROOM-BT-TX v1.6.1 (CB-reset enabled)

Uwagi:
- Jesli inny modul ESP32, najlepiej zostac przy pelnym wgraniu trzema plikami.
- Te pliki sa z jednego, spojnego builda (ten sam czas kompilacji).
